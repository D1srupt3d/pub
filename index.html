Usage Guide

Send Encrypted Email

Import my GPG key and encrypt your message for secure communication.

# Import the key
gpg --import liam-gamache.asc

# Encrypt a message
gpg --encrypt --armor --recipient liam@gamache.dev message.txt

Verify Signatures

Confirm authenticity of signed files and git commits.

# Verify a signed file
gpg --verify file.sig file

# Verify a git commit
git verify-commit <commit-hash>
