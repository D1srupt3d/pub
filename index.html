How to Use

Send me an encrypted email

Import my GPG key and encrypt your message:

# Import the key
gpg --import liam-gamache.asc

# Encrypt a message
gpg --encrypt --armor --recipient liam@gamache.dev message.txt

Verify my signatures

Verify that a signed file or git commit is authentically from me:

# Verify a signed file
gpg --verify file.sig file

# Verify a git commit
git verify-commit <commit-hash>

Only public keys are shared here. Never share private keys.
